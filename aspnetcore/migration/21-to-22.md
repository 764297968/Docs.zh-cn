## <a name="update-target-framework-moniker-tfm"></a><span data-ttu-id="4ec5c-106">更新目标框架名字对象 (TFM)</span><span class="sxs-lookup"><span data-stu-id="4ec5c-106">Update Target Framework Moniker (TFM)</span></span>

<span data-ttu-id="4ec5c-109">面向.NET Framework 的项目可以继续使用大于或等于.NET Framework 4.6.1 版本的 TFM:</span><span class="sxs-lookup"><span data-stu-id="4ec5c-109">Projects targeting .NET Framework may continue to use the TFM of a version greater than or equal to .NET Framework 4.6.1:</span></span>


## <a name="adopt-the-iis-in-process-hosting-model"></a><span data-ttu-id="4ec5c-110">采用的 IIS 进程内承载模型</span><span class="sxs-lookup"><span data-stu-id="4ec5c-110">Adopt the IIS in-process hosting model</span></span>

<span data-ttu-id="4ec5c-111">若要采用[IIS 进程内承载模型](xref:fundamentals/servers/aspnet-core-module#in-process-hosting-model)，添加`<AspNetCoreHostingModel>`属性值为`InProcess`到`<PropertyGroup>`项目文件中：</span><span class="sxs-lookup"><span data-stu-id="4ec5c-111">To adopt the [in-process hosting model for IIS](xref:fundamentals/servers/aspnet-core-module#in-process-hosting-model), add the `<AspNetCoreHostingModel>` property with a value of `InProcess` to a `<PropertyGroup>` in the project file:</span></span>

```xml
<AspNetCoreHostingModel>InProcess</AspNetCoreHostingModel>
```

<span data-ttu-id="4ec5c-112">有关详细信息，请参阅 <xref:host-and-deploy/aspnet-core-module#hosting-model> 。</span><span class="sxs-lookup"><span data-stu-id="4ec5c-112">For more information, see <xref:host-and-deploy/aspnet-core-module#hosting-model>.</span></span>

## <a name="update-net-core-sdk-version-in-globaljson"></a><span data-ttu-id="4ec5c-113">在 global.json 中更新 .NET Core SDK 版本</span><span class="sxs-lookup"><span data-stu-id="4ec5c-113">Update .NET Core SDK version in global.json</span></span>

<span data-ttu-id="4ec5c-114">如果解决方案依靠[global.json](/dotnet/core/tools/global-json)文件，以面向特定.NET Core SDK 版本中，更新其`version`为您的计算机上安装的 2.2 版本的属性：</span><span class="sxs-lookup"><span data-stu-id="4ec5c-114">If your solution relies upon a [global.json](/dotnet/core/tools/global-json) file to target a specific .NET Core SDK version, update its `version` property to the 2.2 version installed on your machine:</span></span>

```json

  "sdk": {
    "version": "2.2.100"
  }
}
```
